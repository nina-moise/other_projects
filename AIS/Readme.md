# AIS - показатели крови и антропометрии элитных спортсменов

[HTML](https://github.com/nina-moise/other_projects/blob/main/AIS/ais_project.html) [ipynb](https://github.com/nina-moise/other_projects/blob/main/AIS/ais_project.ipynb)

## Цель проекта 
Исследовать влияние пола и вида спорта на показатели крови и  безжировую массу тела спортсменов, выявить остальные зависимости. На основе данных построить две модели для машинного обучения для предсказания веса спортсмена на основе его роста, сравнить и выбрать лучшую.

## Результаты 

- Исследовательский анализ данных (EDA) продемонстрировал физиологические различия между полами, специфику видов спорта и взаимосвязь признаков.

- Машинное обучение (ML):  В ходе работы были построены и протестированы две регрессионные модели для предсказания веса спортсмена на основе его роста. 
### Сравнение моделей (Metrics)
| Модель | MAE | R² Score |
| :--- | :---: | :---: |
| Linear Regression | 6.92 кг | 0.60 |
| Random Forest | 7.03 кг | 0.57 |

- Проект продемонстрировал, что для качественного предсказания веса недостаточно только роста. Даны рекомендации по улучшению модели.
